Useful for creating simple progress bars [^1].

Styling is limited, but browser support is good.
### Example

```html
<label for="fuel">Fuel level:</label>
<meter id="fuel" min="0" max="100" low="33" high="66" optimum="80" value="50">
  at 50/100
</meter>
```
